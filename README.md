# Today I Learned (日々の学習記録)

## 概要
千葉県立船橋テクノスクール IoTシステム科での授業内容や、放課後・休日の自主学習でnotebookLMと対話しながら学んだ内容を記録しているリポジトリです。

## 主な記録内容
当リポジトリには、日々のテキストログを中心に以下の内容を格納しています。

* **毎日の学習ログ（今日学んだ内容.txt）**: notebookLMと対話しながら学習したログを記録として残しています。
* **C言語学習**: サーティファイC言語プログラミング能力検定2級に向けた過去問演習や、ポインタ・構造体などの動作確認コード。
* **資格学習ノート**: 基本情報技術者試験（FE）や、デジタル通信に関する学習メモ。
* **その他の技術要素**: SQLの基礎演習、アナログライントレーサー関連の知識整理など。

## 学習のスタンス
私は決して器用なタイプではなく、新しい技術や概念を一度で完璧に理解することは得意ではありません。
しかし、「なぜこのエラーが起きたのか」「裏側でどのような仕組みが動いているのか」をAIツール等を活用して、一つずつ対話しながら、着実に知識を定着させるよう努めています。
華やかな開発物はありませんが、エンジニアを目指して基礎を固めるための積み重ねの記録として残しています。
